A simple and light way to add all useful metadata to your WP theme.

== Description ==
4 metadata types are available : Open Graph, Twitter Card, Dublin Core and HTML.
Light and easy to use.
The plugin is available in FR, EN and NL.

== Installation ==
1. Upload the decompressed simple-metadata-generator.zip to the `/wp-content/plugins/` directory
Or upload it via WP's dashboard : Plugins > Add New
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adjust your settings through the 'Settings' panel menu
